# Proyecto de Control Cinemático para Manipulador Paralelo 2RRR

## Descripción General
Este proyecto se enfoca en el desarrollo y validación de un sistema de control cinemático para un manipulador paralelo tipo 2RRR. El objetivo es optimizar la precisión y eficiencia del manipulador a través de un control adecuado y una calibración precisa.

## Estructura del Proyecto

### Fases del Proyecto
- **Diseño y Planificación**
  - Definición de objetivos y requerimientos.
  - Diseño del manipulador y sus componentes.

- **Desarrollo del Sistema de Control**
  - Implementación de algoritmos de control.
  - Pruebas iniciales y ajustes del software.

- **Montaje y Pruebas Mecánicas**
  - Ensamblaje del manipulador.
  - Pruebas de funcionamiento mecánico y ajustes.

- **Calibración y Validación**
  - Calibración detallada de la cámara y sensores.
  - Validación final del sistema.

### Calibración de la Cámara
- **Preparación**: Asegurar una buena iluminación y un fondo neutro.
- **Procedimiento**:
  - Usar un tablero de calibración en diferentes posiciones.
  - Capturar múltiples imágenes para calibrar.
- **Software**: OpenCV u otros para procesar imágenes.
- **Ajustes**: Corrección de distorsión y alineación con el sistema.

- results/: Resultados de pruebas y reportes.
